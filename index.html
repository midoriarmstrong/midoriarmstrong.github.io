    Contract Developer at Sparhawk Software for the OPC Foundation

    AMQP message subscriber and JSON parser

    I created an AMQP message subscriber in C to subscribe to to data published by an OPC UA Server using the OPC UA PubSub specification, with v0.12.0 of the Apache Qpid Proton library. The subscriber allows users to input the broker address, username, password, and terminus name as command line inputs to subscriber to a message broker. I wrote a full JSON parser for the subscriber, including parsing date/time input into a more readable form.

    (I also slightly adjusted the Qpid Proton release due to a bug with connecting to Azure Service Bus.)
